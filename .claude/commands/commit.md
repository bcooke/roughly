Create a git commit with the current changes.

Follow the git commit workflow:
1. Review changes with `git status` and `git diff`
2. Stage appropriate files
3. Create a descriptive commit message following conventional commits format

Format:
```
<type>: <description>

[optional body]
```

Types: feat, fix, docs, style, refactor, test, chore

**Note**: Do NOT include Claude Code branding, Co-Authored-By trailers, or any AI attribution in commits or PRs.
